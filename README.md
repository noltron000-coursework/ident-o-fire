## Setup
1. `touch .gitmodules`
1. `git submodule add git@github.com:cair/Fire-Detection-Image-Dataset.git data`
1. extract each `.rar` file in the submodule
	- ensure that the directory structure is formatted in this way: `./data/*/*.jpg`
